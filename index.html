Introduction

These visualisations serve to answer Research Question 3 of part 3 of the Data Visualisation exam:

  • Analyse the global rollout of vaccines all over the world. What nations have rolled out vaccinations most effectively?

    • Providing data for this research question - there are three visualistions in total clickcable under the left-side navbar:

  • A vaccination chloropleth map
  • A vaccination lollipop chart
  • Lastly, a donut chart of the most vaccinated nations

    • Key information

    Development of the visualisations was multi-faceted and all the visualisations use:

  • HTML and JavaScript
  • D3.js v.4 and v.5 via: https://d3js.org/
  • Some degree of responsiveness on hover and zoom
  • Some css work for a smoother experience

    • As an open source project each visualisation is based on other developers code as follows:

    The vaccination chloropleth:

  • Based on code developed by Mikael Koutero and sourced at: https://bl.ocks.org/eetuko/4535086c3fabe76a173b432c44b254c6
  • The use of the topojson to load the shape file
  • The addition of a legend using Susie Lu's legend: https://d3-legend.susielu.com/

    • The vaccination lollipop chart:

  • Based on code developed by Yan Holtz and sourced at: https://www.d3-graph-gallery.com/graph/lollipop_animationStart.html

    • The most vaccinated nations donut chart:

  • Based on code developed by KJ Schmidt and sourced at: https://medium.com/@kj_schmidt/making-an-animated-donut-chart-with-d3-js-17751fde4679

    • About the data

    The data for the three visualisations comes from a single source:

  • The data is sourced from March/June OWID data via: https://github.com/owid/covid-19-data/tree/master/public/data

    • Significant data amalgamating, parsing and cleaning of the data was written in Java to create custom CSV and JSON files:

  • The JSON-Simple library was sourced and used to parse clean and amalgamate data via: http://www.java2s.com/Code/Jar/j/Downloadjsonsimple11jar.htm - this is accessible under the respective GitHub folders

    • Vaccination chloropleth calculations

    The vaccination chloropleth maps display divided geographical areas or regions are coloured, shaded or patterned in relation to the variable selected by the user.

    Each vaccination chloropleth comes with 4 optional metrics choosable via the dropdown (all mathematic calculations are rounded to 3 decimal places):

  • Total vaccinations - Total number of vaccinations used by each nation for their population.
  • People vaccinated - Total number of people to recieve at least one vaccination within each nation.
  • Total vaccinated proportionally - To gain a proportional metric of each nation's vaccination rollout with: (Total vaccinated in current nation / Total vaccinated worldwide) x 100
  • People vaccinated proportionally - To gain a proportional metric of each nation's vaccination rollout with: (People vaccinated in current nation / People vaccinated worldwide) x 100

    • Other interactive features of the vaccination chloropleths include:

  • Hover on countries for an accurate metric and pop up of underlying value
  • Furthermore the maps are fully dynamic enabling zoom via CTRL and SCROLL Windows or CMD and SCROLL Mac
